# Война и мир Льва Толстого 
Главные герои, представители высшего дворянского сословия, оказываются в плену любовного треугольника. Их непростые отношения развиваются на фоне драматических событий войны 1812 года: пережив множество испытаний, каждый из них переосмысливает свою жизнь.
